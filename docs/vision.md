# Smart Drill Vision

## プロジェクト概要

Smart Drill は、家庭学習を支援するAI対応学習プラットフォームである。

ChatGPTを活用して教材を作成し、Smart Drill
が学習履歴を管理・分析することで、子ども一人ひとりに最適な復習を提供する。

------------------------------------------------------------------------

# 目的

単に問題を解くだけではなく、

**「忘れない学習」**

を実現する。

------------------------------------------------------------------------

# コンセプト

## AIは教材を作る

教材は ChatGPT を利用して作成する。

-   塾プリント
-   学校ワーク
-   PDF
-   画像

などから Smart Drill Package（SDP）を生成する。

## Smart Drillは学習を管理する

Smart Drill は次の情報を管理する。

-   学習履歴
-   理解度
-   忘却曲線
-   苦手分析

## 問題と学習履歴は分離する

問題データと学習データは別管理とする。

これにより、

-   教材追加
-   教材共有
-   複数教科対応

を容易にする。

## AIは教材作成支援を担当する

学習中の判定は Smart Drill が行う。

AIは次を担当する。

-   教材生成
-   類題生成
-   ヒント作成
-   解説生成

------------------------------------------------------------------------

# 対象

最初は

-   中学生
-   英語

を対象とする。

将来的には

-   数学
-   国語
-   理科
-   社会
-   英検
-   TOEIC

などへ拡張する。

------------------------------------------------------------------------

# 開発方針

実装より設計を優先する。

新機能を追加する前に、

-   拡張性
-   保守性
-   ユーザー体験

を確認する。

------------------------------------------------------------------------

# プロジェクトの理念

子どもの学力は、

「どれだけ授業を受けたか」ではなく、

「どれだけ適切なタイミングで復習できたか」

で大きく変わる。

Smart Drill
は、AIと学習データを活用し、家庭で無理なく復習を継続できる環境を提供する。

------------------------------------------------------------------------

# ゴール

家庭で毎日使えるAI対応学習プラットフォームを作る。
