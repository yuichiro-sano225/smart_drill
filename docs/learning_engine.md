# Smart Drill Learning Engine

## 目的

学習エンジンは「どの問題を、いつ出題するか」を決定する。
目標は、最小の学習時間で最大の定着率を実現することである。

------------------------------------------------------------------------

# 基本方針

出題順は以下の要素を総合して決定する。

1.  復習予定日（忘却曲線）
2.  苦手度
3.  テスト日までの残日数
4.  問題の重要度
5.  最近解いたかどうか

これらをスコア化し、優先度の高い問題から出題する。

------------------------------------------------------------------------

# 学習サイクル

新規学習 ↓ 問題を解く ↓ 正誤を記録 ↓ 次回復習日を決定 ↓
おすすめ問題へ反映

------------------------------------------------------------------------

# 忘却曲線

初期値（変更可能）

  回数          次回復習
  ------------- ----------
  初回正解      1日後
  2回連続正解   3日後
  3回連続正解   7日後
  4回連続正解   14日後
  5回連続正解   30日後

※ この日数は設定画面から変更できるようにする。

------------------------------------------------------------------------

# 苦手補正

間違えた問題は優先度を上げる。

-   直近で間違えた問題
-   同じ問題を繰り返し間違えている
-   長期間正解できていない

これらは通常より早く再出題する。

------------------------------------------------------------------------

# テスト補正

テスト日が設定されている場合は、 対象単元の優先度を自動的に上げる。

テストが近づくほど重みを大きくする。

------------------------------------------------------------------------

# 今日のおすすめ

「今日のおすすめ」は以下を組み合わせて作成する。

-   復習期限が来た問題
-   苦手問題
-   テスト範囲
-   新規問題（必要に応じて）

単元を意識せず学習できることを目標とする。

------------------------------------------------------------------------

# 設計方針

アルゴリズムは固定値ではなく、
重みや日数を設定画面から変更できる設計とする。

将来的にはAIによる最適化も検討する。
